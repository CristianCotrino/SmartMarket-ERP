# SmartMarket ERP

SmartMarket ERP es un sistema de planificación de recursos empresariales (ERP) diseñado para la gestión de un supermercado.

## Objetivo

Optimizar los procesos de inventario, ventas, compras, empleados y generación de reportes mediante una solución estructurada basada en metodología Scrum.

## Metodología

El proyecto fue desarrollado siguiendo la metodología ágil Scrum, incluyendo:

- Product Backlog
- Sprint Planning
- Estimación con Planning Poker
- Definition of Done
- Plan de pruebas
- Diagramas UML

## Módulos Principales

- Gestión de Inventario
- Gestión de Ventas
- Gestión de Compras
- Gestión de Empleados
- Reportes y Análisis

## Diagramas

El proyecto incluye:

- Árbol de Descomposición Funcional
- Diagrama UML de Casos de Uso
- Diagrama UML de Clases
